# 🏥 Hospital Management System Analysis Using SQL

## 📖 Project Overview

This project demonstrates how SQL can be used to analyze hospital operations, patient management, financial performance, medical inventory, and healthcare resource utilization using a relational database in MySQL.

The project answers real-world healthcare business questions that support hospital administrators, finance departments, medical staff, and operations managers in making informed, data-driven decisions. Through SQL queries, the project uncovers operational trends, identifies performance gaps, and generates actionable insights to improve patient care and hospital efficiency.

---

# 🎯 Project Objectives

The primary objectives of this project are to:

- Monitor patient admissions and discharges.
- Analyze hospital utilization and patient flow.
- Identify the most common diagnoses and treatments.
- Evaluate doctor and department performance.
- Monitor medicine inventory and stock value.
- Analyze hospital revenue and billing performance.
- Assess payment trends and outstanding balances.
- Improve operational efficiency using healthcare data.
